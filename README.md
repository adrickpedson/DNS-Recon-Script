# Dns-Reconnaissance

A simple shell script to collect DNS records of a domain created as a part of learning shell scripting

## Current features
  ```sh
  - Can collect CNAME, A, AAAA, MX, TXT records of the domain
  ```
  
## Running
   ```sh
   $ chmod 755 dnsrecon.sh
   $ ./dnsrecon.sh
   ```
